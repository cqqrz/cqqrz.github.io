QSL Card Storage Box

Storage boxes for QSL cards (3.5 x 5.5) and post cards (4 x 6) including a special 3905 Century Club edition box with band dividers.

  • QSL card box (3.5 x 5.5): qsl-card-box.stl qsl-card-box-lid.stl
  • Post card box (4 x 6): post-card-box.stl post-card-box-lid.stl
  • 3905 Century Club card box (4 x 6): 3905-card-box.stl post-card-box-lid.stl
  • Post card box dividers (for 4 x 6 boxes): divider-20m.stl divider-40m.stl divider-80m.stl divider-160m.stl divider-digital.stl

600 Ohm Ladder Line Spreader

These spreaders are for DIY build of 600 Ohm ladder lines using the stranded 14-AWG wire that are common in the USA. The wires simply snap into the end and are spaces 5.5 inches apart. Use ASA filament for UV and weather resistance.

  • STL file of the spreader: 600-ohm-ladder-line-spreader.stl
  • G-Code for the Prusa MK3S with 16 spreaders for ASA: 600-ohm-ladder-line-spreader_0.2mm_ASA_MK3S_3h23m.gcode
