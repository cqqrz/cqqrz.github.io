Yaesu Bluetooth CAT Interface

I've seen a few projects that add bluethooth support for the CAT interface. I'm mainly looking at using my android phone to control the radio via bluetooth. For computer work I most likely still use a wired CAT cable.

At this point I'm still in the planning stage, trying to figure out how to hook up the HC-05 bluetooth module to the mini-DIN connector.
